# 💰 Personal Finance Tracker

## 📖 Description

Personal Finance Tracker is a Python application that helps users manage their daily income and expenses. It includes both a Console Interface and a Tkinter GUI where users can add, view, delete, and summarize financial transactions. All transactions are automatically saved in a JSON file.

---

## ✨ Features

- Add Income and Expense transactions
- View all saved transactions
- Delete transactions
- Calculate Total Income
- Calculate Total Expenses
- Calculate Remaining Balance
- Category-wise Expense Summary
- Search transactions by category (Console)
- Automatic saving using JSON
- Load previous records automatically
- User-friendly Tkinter GUI
- Error handling for invalid inputs
- Recursive functions for calculations
